Я обчислила означений інтеграл функції ( f(x) = x^2 ) на відрізку [0, 2]
за допомогою методу Монте-Карло і порівняла результат з функцією quad
із підмодуля scipy.integrate.

Результати вийшли дуже близькими один до одного, різниця невелика.
Якщо збільшувати кількість випадкових точок у методі Монте-Карло,
наближене значення ще краще підходить до значення, яке повертає quad.

Отже, метод Монте-Карло дає хорошу наближену оцінку інтеграла,
а quad можна використовувати як еталон для перевірки точності.
